Xifrat Juli Cèsar

L’objectiu d’aquesta activitat és utilitzar alguna de les funcions predefinides del llenguatge JavaScript. Podeu utilitzar les funcions: chatAt(), indexOf(), alert(), length(), prompt() i toUpperCase().

El dictador Juli Cèsar utilitzava un codi quan volia mantenir en secret un missatge. El xifrat consistia a substituir la primera lletra de l’alfabet (A) per la quarta (D) i així successivament amb les altres, és a dir, B, per la cinquena (I) la tercera (C) per la sisena (F)…

L’alfabet llatí que utilitzava Juli Cèsar constava de 21 lletres, per tant la substitució de lletres per xifrar o desxifrar missatges queda descrita com segueix: 

Alfabet original : A B C D E F G H I K L M N O P Q R S T V
Alfabet xifrat   : D E F G H I K L M N O P Q R S T V A B C

Aquest tipus de xifrat és un codi de rotació, on la clau de rotació és 3. Escriu un programa que permeti xifrar utilitzant un codi de rotació. La clau serà una variable del programa.

Podeu provar aquesta activitat a: https://codepen.io/ioc-daw-m06/pen/oNLEJLm?editors=0010 

  1. let alfabet = "ABCDEFGHIKLMNOPQRSTV";
  2. let clau = 3;
  3.  
  4. let text = prompt("Escriu el text a xifrar").toUpperCase();
  5. let textxifrat = "";
  6. for (let i = 0; i < text.length; i++) {
  7.   let charorig = text.charAt(i);
  8.   let poscharxifrat = (alfabet.indexOf(charorig) + clau) % alfabet.length;
  9.   textxifrat = textxifrat + alfabet.charAt(poscharxifrat);
  10. }
  11. alert(`El resultat és el següent: ${textxifrat}`);

Com podeu observar, primer es demana a l’usuari que introdueixi quin text vol xifrar amb la funció propmt(). Després, s’agafa cadascuna de les lletres, mitjançant la funció chatAt(), i es calcula la posició de la lletra dintre del alfabet amb la funció indexOf().

Per saber quina seria la lletra xifrada s’aplica l’algoritme Juli Cèsar. A la posició de la lletra dins de l’alfabet se suma la clau (3) i al resultat s’aplica el mòdul de la longitud de l’alfabet per tornar a començar en el cas que donés un número superior a la longitud de l’alfabet.

Totes les lletres xifrades es guarden en una variable per mostrar el resultat final.

Contrasenya segura

L’objectiu d’aquesta activitat és que realitzeu un programa utilitzant les funcions predefinides del llenguatge JavaScript. En concret, es demana que realitzeu un programa que verifiqui si una contrasenya és segura.

Per tal que sigui segura ha de complir les següents regles:

  • Ha de contenir almenys una vocal
  • No ha de tenir tres vocals consecutives o tres consonants consecutives
  • No ha de tenir dues ocurrències consecutives de la mateixa lletra, excepte per ‘ee’ o ‘oo’
  • La longitud mínima és de 6 caràcters i la màxima de 10
  • Els espais en blanc no estan permesos

Podeu provar aquesta activitat a: https://codepen.io/ioc-daw-m06/pen/LYZQMxq?editors=0010 

  1. let pass = "rtiseewaf";
  2.  
  3. let contvocals = 0;
  4. let contconsonants = 0;
  5. let almenysvocal = 0;
  6. let segur = true;
  7. if (pass.length > 10 && pass.length < 6) {
  8.   alert(
  9.     "Password no segur, la mida del password ha de ser més gran que 6 i menys que 10"
  10.   );
  11. } else {
  12.   for (let i = 0; i < pass.length; i++) {
  13.     if (
  14.       pass.charAt(i) == "a" ||
  15.       pass.charAt(i) == "u" ||
  16.       pass.charAt(i) == "e" ||
  17.       pass.charAt(i) == "o" ||
  18.       pass.charAt(i) == "i"
  19.     ) {
  20.       contvocals++;
  21.       contconsonants = 0;
  22.       almenysvocal++;
  23.     } else {
  24.       contconsonants++;
  25.       contvocals = 0;
  26.     }
  27.     if (contvocals == 3 || contconsonants == 3) {
  28.       segur = false;
  29.       alert("Password no segur, 3 vocals o 3 consonants consecutives");
  30.     }
  31.     if (
  32.       pass.charAt(i) == pass.charAt(i + 1) &&
  33.       pass.charAt(i) != "e" &&
  34.       pass.charAt(i) != "o"
  35.     ) {
  36.       segur = false;
  37.       alert("Password no segur, no pot haver 2 consonants o vocals iguals");
  38.     }
  39.     if (pass.charAt(i) == " ") {
  40.       segur = false;
  41.       alert("Password no segur, no pot contenir espais");
  42.     }
  43.   }
  44.   if (segur && almenysvocal > 0) {
  45.     alert("Password Segur");
  46.   } else {
  47.     alert("Password no segur, no hi ha cap vocal.");
  48.   }
  49. }

En aquesta exercici, la contrasenya no es demana a l’usuari sinó que està guardada en una variable del programa. Utilitzant aquesta variable es miren totes les condicions que estableixen que una contrasenya és segura.

Primer de tot es mira que la longitud de la contrasenya estigui dins dels paràmetres de seguretat. Una vegada s’ha comprovat, es mira caràcter a caràcter si compleix les altres restriccions de seguretat. Per exemple, hi ha un comptador de vocals i un altre de consonants que compten quantes vocals o consonants consecutives hi ha. En el cas que hi hagi 3 o més vocals o consonants consecutives es considera insegur. Si es troba que hi ha dos vocals o consonants consecutives iguals també es considera insegur, a no ser que sigui una e o una o.

També es comprova que la contrasenya no contingui espais.

Bender

L’objectiu d’aquesta activitat és que utilitzeu funcions predefinides del llenguatge JavaScript implementant la sobrecàrrega d’un mètode d’un objecte.

Tenim un robot anomenat “Bender”, que li agrada dividir cadenes. Implementeu el mètode divideix() de diferents maneres utilitzant el número de paràmetres que s’envien al mètode per definir el seu comportament. 

  1. bender = {}

El seu comportament és el següent:

  • Si li passen només 1 cadena: divideix aquesta cadena per la meitat i retorna la primera part.
  • Si li passen dues cadenes: primer divideix la primera cadena per la meitat (A,B). Deprés la segona cadena, també la divideix per la meitat (C,D). Retorna una string concatenant les parts de la següent manera: CBDA
  • Si li passen dues cadenes i un número: concatena a la primera cadena tants caràcters de la segona cadena com digui el número. Si el número és més gran que la longitud de la cadena tornarà a començar a concatenar des del principi fins que s’arribi al número indicat.

Podeu provar aquesta activitat a: https://codepen.io/ioc-daw-m06/pen/mdEXamy?editors=0010 

  1. function afegirMetode(objecte, nom, funcio) {
  2.   let old = objecte[nom];
  3.   objecte[nom] = function () {
  4.     if (funcio.length == arguments.length) return funcio.apply(this, arguments);
  5.     else if (typeof old == "function") return old.apply(this, arguments);
  6.   };
  7. }
  8.  
  9. bender = {};
  10.  
  11. //Funció amb 1 paràmetre. Retorna la primera meitat de la cadena.
  12. afegirMetode(bender, "divideix", function (a) {
  13.   let meitat = a.length / 2;
  14.   return a.substr(0, meitat);
  15. });
  16.  
  17. //Funció amb 2 paràmetres. Retorna un string construït amb trossos de les cadenes d'entrada.
  18. afegirMetode(bender, "divideix", function (cadena1, cadena2) {
  19.   let meitat1 = cadena1.length / 2;
  20.   let meitat2 = cadena2.length / 2;
  21.   let a = cadena1.substr(0, meitat1);
  22.   let b = cadena1.substr(meitat1);
  23.   let c = cadena2.substr(0, meitat2);
  24.   let d = cadena2.substr(meitat2);
  25.   return c + b + d + a;
  26. });
  27.  
  28. afegirMetode(bender, "divideix", function (cadena1, cadena2, nombre) {
  29.   if (!isNaN(nombre)) {
  30.     let cont = 0;
  31.     let pos = 0;
  32.     let resultat = cadena1;
  33.     while (cont < nombre) {
  34.       if (pos > cadena2.length - 1) {
  35.         pos = 0;
  36.       }
  37.       resultat += cadena2.charAt(pos);
  38.       cont++;
  39.       pos++;
  40.     }
  41.     return resultat;
  42.   }
  43. });
  44.  
  45. //Exemple de funcionament:
  46. document.write(`1 paràmetre: ${bender.divideix("HOLA")}<br>`);
  47. document.write(`2 paràmetres: ${bender.divideix("HOLA", "ADEU")}<br>`);
  48. document.write(`3 paràmetres : ${bender.divideix("HOLA", "ADEU", 5)}<br>`);

Per realitzar la sobrecàrrega de funcions s’utilitza la funció afegirMetode(). La funció sobrecarrega un mètode d’un objecte. Necessita un objecte, el nom del mètode al qual s’aplica la sobrecàrrega i la funció de sobrecàrrega.

Es crida a la funció afegirMetode() tres vegades, amb tres funcions diferents que reben paràmetres diferents. La primera vegada s’envia una funció amb 1 paràmetre. Aquesta funció divideix una cadena per la meitat i retorna la primera part.

La segona vegada retorna una cadena construïda amb trossos de les dues cadenes d’entrada i la tercera vegada, concatena a la primera cadena tants caràcters de la segona cadena com digui el número passat per paràmetre.

Cens

Ens han demanat la realització d’un cens per províncies. Per implementar-lo cal que l’aplicació ens demani en primer lloc el nom de cada província i el seu nombre d’habitants i que les províncies s’afegeixin a una estructura de dades que ha de permetre:

  • afegir noves províncies.
  • eliminar una província per nom.

Addicionalment caldrà implementar una funció per mostrar el llistat actualitzat de províncies.

També caldrà que l’aplicació, un cop afegida l’última província i mostrat el llistat de les províncies, demani el nom d’una província que caldrà esborrar i mostri el llistat després d’aquesta eliminació.

Utilitzeu el tipus de col·lecció més adient per aquesta tasca. Cal tenir en compte que els noms de les províncies introduïdes serà únic. No cal fer gestió d’errors d’entrada.

A continuació podeu trobar la solució proposada per aquesta activitat. Fixeu-vos que s’ha fet servir un map, ja que sabem que els noms de les províncies és únic i aquest tipus de col·lecció ens permet afegir-hi elements, eliminar-los i consultar-los fàcilment.

Podeu provar aquesta activitat a: https://codepen.io/ioc-daw-m06/pen/vYKdbpm?editors=1011 

  1. <h1>Instruccions</h1>
  2. <p>Introdueix províncies i els seus habitants.</p>
  3. <p>Per sortir has de premer enter quan pregunti el nom de la nova província.</p>
  4. <h1>Llistat de províncies</h1>
  5. <script>
  6.   let provincies = new Map();
  7.   let sortir = false;
  8.  
  9.   function mostrarProvincies(provs) {
  10.     let total = 0;
  11.     for (let [nom, habitants] of provs) {
  12.       document.write(`${nom}: ${habitants} habitants<br>`);
  13.       total += habitants;
  14.     }
  15.     document.write(`Habitants totals: ${total}`);
  16.   }
  17.   do {
  18.     let nom = prompt("Nom província:");
  19.     let habitants;
  20.     if (!nom) {
  21.       sortir = true;
  22.     } else {
  23.       habitants = Number(prompt("Habitants:"));
  24.     }
  25.     if (nom && habitants != NaN) {
  26.       provincies.set(nom, habitants);
  27.     }
  28.   } while (!sortir);
  29.   mostrarProvincies(provincies);
  30.   document.write(`<h1>Eliminació</h1>`);
  31.   let eliminar = prompt(`Introdueix el nom d´una província per eliminar`);
  32.   if (provincies.has(eliminar)) {
  33.     provincies.delete(eliminar);
  34.     document.write(`Eliminat: ${eliminar}`);
  35.   } else {
  36.     document.write(`<p><b>L´element no existeix</b></p>`);
  37.   }
  38.   document.write(`<h1>Llistat final</h1>`);
  39.   mostrarProvincies(provincies);
  40. </script>

Gestió d'alumnes

Ens han demanat implementar un sistema de gestió d’alumnes molt simple: només cal controlar la addició i l’eliminació. També cal poder llistar tots els alumnes actualment a la llista.

Us indiquen que ha d’utilitzar-se un set que ja tenen definit: 

  1. let alumnes = new Set();

Per tant, haureu d’implementar tres funcions:

  • afegirAlumne(nom): afegeix un nou alumne.
  • eliminarAlumne(nom): elimina l’alumne de la llista.
  • llistarAlumnes(): mostra el llistat d’alumnes actual.

Tingueu en compte que per aquesta activitat es considera que el nom de cada alumne és únic.

A continuació podeu trobar la solució proposada per aquesta activitat amb la demostració del seu correcte funcionament.

Podeu provar aquesta activitat a: https://codepen.io/ioc-daw-m06/pen/vYKdMgo?editors=0012 

  1. let alumnes = new Set();
  2.  
  3. function afegirAlumne(nom) {
  4.   alumnes.add(nom);
  5. }
  6.  
  7. function eliminarAlumne(nom) {
  8.   alumnes.delete(nom);
  9. }
  10.  
  11. function llistarAlumnes() {
  12.   console.log("Alumnes:")
  13.   for (let nom of alumnes) {
  14.     console.log(nom);
  15.   }
  16. }
  17.  
  18. afegirAlumne('Pere');
  19. afegirAlumne('Maria');
  20. afegirAlumne('Ramon');
  21. llistarAlumnes();
  22.  
  23. eliminarAlumne('Ramon');
  24. listarAlumnes();

Taula de continguts: Ofereix una vista general de l'estructura del mòdul, que us facilitarà la navegació a través dels seus continguts.
Opcions de format: Permet configurar el format de lectura a partir de les vostres preferències, modificant la mida de la lletra, el color del fons, l'amplada de la pàgina o l'ocultació dels continguts complementaris, entre d'altres.
Imprimir: Envia el contingut seleccionat a la impressora.
Prefereits: Permet desar aquelles parts del contingut a les que us interessi accedir en un moment posterior; us permetrà anar creant un repositori personalitzat de les seccions que considereu més rellevants. Un cop l'hageu començat a fer servir se us mostrarà a sota un comptador que us informarà del nombre de preferits que s'hagin emmagatzemat fins al moment.
Comptador i llistat de preferits: Indica el nombre de preferits que s'han desat fins el moment; clicant damunt seu accedireu al llistat de preferits, i a través d'aquest podreu accedir directament als continguts que hageu emmagatzemat.
Ajuda: Aquesta opció activa la informació de les funcionalitats del web tot situant el punter del ratolí al damunt dels diferents ítems.
Mostrar/Ocultar barra lateral: Deshabilita la barra d'opcions, permetent la seva recuperació quan es desitgi.
Cerca: Introduïu les paraules clau sobre aquells conceptes que desitgeu localitzar en els continguts del mòdul. També podeu excloure un terme de la cerca tot afegint-hi el signe “-” al davant.
Capçalera: Indica l'apartat o secció que es mostra en pantalla. Tingueu present que les capçaleres de segon, tercer i quart nivell també es poden desar com a marcadors.
