Dibuixar un triangle (funcions niades)

L’objectiu de l’activitat és crear una funció JavaScript que utilitzi funcions niades per dibuixar un triangle d’asteriscs.

A l’activitat ha d’aparèixer una funció que donat un número enter (‘num’) dibuixi un triangle rectangle, on cadascuna de les cares del triangle ha de tenir ‘num’ asteriscs. Per realitzar aquesta activitat s’ha d’utilitzar una estructura niada de funcions.

Podeu provar aquesta activitat a: https://codepen.io/ioc-daw-m06/pen/gOMvdEL?editors=0010

El codi de la solució és: 

  1. function triangle(num) {
  2.   for (let i = 1; i <= num; i++) {
  3.     dibuixarLinia(i);
  4.     document.write("<br>");
  5.   }
  6.  
  7.   function dibuixarLinia(rep) {
  8.     for (let j = 0; j < rep; j++) {
  9.       document.write("*");
  10.     }
  11.   }
  12. }
  13. triangle(5);

Per crear una estructura de funcions niades s’ha decidit crear dues funcions. La funció triangle() que és la funció principal i la funció Dibuixarlinia() que és la funció que dibuixa cadascuna de les files del triangle.

La funció principal triangle() té un bucle per dibuixar cadascuna de les línies del triangle. La funció dibuixarLinia() dibuixa tants asteriscs com repeticions li passem per paràmetre. El número de repeticions, és a dir, d’asterics, correspon amb el número de línia que s’està dibuixant. El resultat de l’execució del codi anterior és: 

  1. *
  2. **
  3. ***
  4. ****
  5. *****

Pàrquing (memoize)

L’objectiu de l’activitat és crear una funció que memoritzi els valors calculats prèviament.

A l’activitat ha d’aparèixer una funció que memoritzi els diners que ha de pagar un conductor segons el temps que ha estat estacionat el cotxe en el pàrquing. El temps (minuts) que ha estacionat un cotxe s’envia per paràmetre a la funció. Cada minut que un cotxe es troba estacionat en el pàrquing paga 0.05 cèntims.

Podeu provar aquesta activitat a: https://codepen.io/ioc-daw-m06/pen/eYzVLwb?editors=0010 

  1. function parquing(temps) {
  2.   if (!parquing.cache) parquing.cache = {};
  3.   if (parquing.cache[temps] != null) {
  4.     return `Cache: ${parquing.cache[temps]}`;
  5.   }
  6.   let diners = temps * 0.05;
  7.   parquing.cache[temps] = diners;
  8.   return `Calculat: ${diners}`;
  9. }
  10. document.write(parquing(60) + " euros<br>");
  11. document.write(parquing(60) + " euros<br>");
  12. document.write(parquing(120) + " euros<br>");
  13. document.write(parquing(60) + " euros<br>");
  14. document.write(parquing(120) + " euros<br>");

La funció parquing() rep el temps, en minuts, que un cotxe ha estacionat dins del pàrquing. La funció comprova si ja havia calculat el valor anteriorment i així s’estalvia tornar a calcular-ho.

Primer comprova que estigui creada la cache. Si s’ha creat, comprova que s’hagi calculat prèviament els diners segons el temps que ha estat estacionat el cotxe. En el cas que s’hagi calculat, es retorna aquest càlcul i s’acaba la funció.

Si és la primera vegada que hi ha estat un cotxe durant aquest temps llavors es calcula els diners que costa el seu estacionament. Es retorna els diners al mateix temps que es guarda en memòria de la funció. El resultat de l’execució és el següent: 

  1. Calculat: 3 euros
  2. Cache: 3 euros
  3. Calculat: 6 euros
  4. Cache: 3 euros
  5. Cache: 6 euros

Rellotge (setTimeout i funcions definides pel programador)

L’objectiu de l’activitat és crear un rellotge utilitzant funcions definides pel programador i la funció setTimeout().

A l’activitat ha d’aparèixer una funció que escrigui l’hora actual per pantalla. Cada mig segon s’ha d’actualitzar per donar la sensació que realment estem visualitzant un rellotge. Aquest rellotge ha de tenir el format següent: HH:MM:SS

Podeu provar aquesta activitat a: https://codepen.io/ioc-daw-m06/pen/BazYqBd?editors=0010 

  1. function rellotge() {
  2.   let temps = new Date();
  3.   let h = temps.getHours();
  4.   let m = temps.getMinutes();
  5.   let s = temps.getSeconds();
  6.   h = afegirZeros(h);
  7.   m = afegirZeros(m);
  8.   s = afegirZeros(s);
  9.   document.body.innerHTML = `${h}:${m}:${s}`;
  10.   setTimeout(rellotge, 500);
  11. }
  12.  
  13. function afegirZeros(i) {
  14.   return String(i).padStart(2, '0');
  15. }
  16.  
  17. rellotge();

La funció principal es diu rellotge(). Aquesta funció calcula l’hora, els minuts i els segons actuals. Aquests càlculs s’emmagatzemen en variable per tal de mostrar-los com ens diu l’enunciat.

El primer problema amb què ens trobem és que sempre hem de mostrar dos números, encara que les hores, els minuts o els segons calculats només hi hagi un dígit. Llavors s’ha d’afegir un ‘0’ al començament,

la solució és convertir el nombre en una cadena de text i fer servir el mèdote padStart() de l’objecte String que ens permet omplir la mida de la cadena amb el caràcter indicat.

Una vegada s’ha solucionat el problema ja el podem escriure al document.

S’ha de tornar a repetir aquesta funció cada mig segon per tal de poder mostrar correctament l’hora. Llavors es fa ús de la funció setTimeOut() per programar la següent execució de la funció. El resultat de l’execució és el següent: 

  1. 00:03:06

Rellotge (funció de fletxa)

L’objectiu d’aquesta activitat és crear un rellotge mitjançant una funció de fletxa com a callback de la funció setInterval().

A l’activitat ha d’aparèixer una funció que escrigui l’hora actual per pantalla. Cada mig segon s’ha d’actualitzar per donar la sensació que realment estem visualitzant un rellotge. Aquest rellotge ha de tenir el format següent: HH:MM:SS

Podeu provar aquesta activitat a: https://codepen.io/ioc-daw-m06/pen/BazYqjm?editors=0010 

  1. function afegirZeros(i) {
  2.   return String(i).padStart(2, "0");
  3. }
  4.  
  5. setInterval(() => {
  6.   let temps = new Date();
  7.   let h = afegirZeros(temps.getHours());
  8.   let m = afegirZeros(temps.getMinutes());
  9.   let s = afegirZeros(temps.getSeconds());
  10.   document.body.innerHTML = `${h}:${m}:${s}`;
  11. }, 500);

Com es pot apreciar, en lloc d’indicar el nom de la funció com a paràmetre s’utilitza una funció de fletxa: () ⇒ {cos}'', i aquesta serà invocada automàticament cada 500 ms. Aquest és un cas d’ús molt habitual per les funcions de fletxa, ja que es tracta d’una funció que no cal invocar-la des de cap altre punt del programa i es fa servir com a callback d’una altra funció.

Processador d'accions (propagació)

Es vol crear una funció que permeti processar accions que seran passades com objectes declarats literalment amb la següent estructura: 

accio = {
  font: "font",
  accio: "acció",
  objectiu: "objectiu"
}

L’objectiu és opcional i el seu valor per defecte ha de ser desconegut (es recomana utilitzar paràmetres amb nom).

A banda d’aquestes propietats els objectes poden incloure altres propietats però no cal tenir-les en compte.

Per demostrar el correcte funcionament la funció només cal que mostreu per la consola un missatge indicant la font, l’acció i l’objectiu.

A continuació podeu veure la solució proposada. S’ha fet servir la desestructuració de l’objecte per extreure els paràmetres i assignar el valor per defecte.

Podeu provar aquesta activitat a: https://codepen.io/ioc-daw-m06/pen/bGeLmKx?editors=0012 

  1. function processar({ font, accio, objectiu = "desconegut" }) {
  2.   console.log(`${font} executa ${accio} sobre ${objectiu}`);
  3. }
  4.  
  5. let menjar = {
  6.   font: "Pere",
  7.   accio: "menjar",
  8.   objectiu: "pastanaga"
  9. };
  10.  
  11. let caminar = {
  12.   font: "Maria",
  13.   accio: "caminar",
  14. };
  15.  
  16. processar(menjar);
  17. processar(caminar);

Fixeu-vos que l’execució és correcta tant si es passen paràmetres de més (en el cas de menjar) com si no es passa l’objectiu, ja que s’assigna un valor per defecte a aquest paràmetre.

Divisió sencera (desestructuració)

S’ha de crear una funció que retorni el resultat d’una divisió sencera com un objecte distingint entre el quocient i el mòdul, per exemple: 

Divisió de 4 entre 3:

resultat = {
  quocient: 1,
  modul: 3
}

Es demana també una demostració del funcionament assignant automàticament els valors de quocient i modul a dos variables amb el mateix nom.

La solució proposada és la següent: 

  1. function dividir(dividend, divisor) {
  2.   let quocient = Math.floor(dividend / divisor);
  3.   let modul = divisor % dividend;
  4.  
  5.   return {
  6.     quocient: quocient,
  7.     modul: modul
  8.   };
  9. }
  10.  
  11. let dividend = 4;
  12. let divisor = 3;
  13. let { modul, quocient } = dividir(dividend, divisor);
  14. console.log(`${dividend} entre ${divisor}: quocient ${quocient} i mòdul ${modul}`);

Podeu provar aquesta activitat a: https://codepen.io/ioc-daw-m06/pen/dyXdQpd?editors=0012

Fixeu-vos que s’ha aprofitat la propagació per assignar automàticament a les variables modul i quocient el valor de les propietats de l’objecte retornat per la funció dividir.
